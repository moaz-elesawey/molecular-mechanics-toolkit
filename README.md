# Molecular Mechanics Toolkit
A toolkit built in python to model `xyz` files and extract the **bonds**, **angles**, **out of planes**
and **torsion**.

It also can compute the energy if provided with a special file type I called it `mxyz`
which stands for `modified xyz` which contains also the `charge` in the last column
and the `symbol` column is replaced with the `atomic type` which is the symbol of the 
element followed by it's type as `CA` and `HA` which denotes a Hydrogen atom and Carbon
atom in atromatic ring.

those `atomic types` is used in order to get the parameters for the energy calculation
from the **Amber99 Force Fields** which I will provide it as well as the **Charm27** and **Oplsaa**

Now the **Energy** calculations is only available for `mxyz` files not for `xyz`


## Help
```sh
$ python run.py --help
```

```
usage: Program that used to model xyz file of chemical
       compounds and calculate the bonds, angles, torsions
       and total energy involved in it.

optional arguments:
  -h, --help            show this help message and exit
  --mxyz MXYZ, --minput MXYZ
                        path to the mxyz file
  --xyz XYZ, --input XYZ
                        path to the xyz file
  --out OUT, --output OUT
                        path to the output file
```

## Sample Run
```sh
$ python run.py --mxyz data/mxyz/benzene.mxyz
```
